# Vinted Clone – Aplicatie Web (.NET 10)

Aplicatie web de tip **Vinted**, dezvoltata pentru vanzarea si cumpararea articolelor second hand.  
Proiect realizat folosind **.NET 10** pentru backend si **HTML/CSS/JavaScript** pentru frontend.

---

## 🛠 Tehnologii utilizate
- .NET 10
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML5
- CSS3
- JavaScript
- Bootstrap

---

## ⚙️ Functionalitati
- Autentificare si inregistrare utilizatori
- Gestionare profil utilizator
- Adaugare, editare si stergere produse
- Afisare produse pe categorii
- Upload imagini produse
- Sistem de autorizare pe roluri
- Validari formular (client + server)
